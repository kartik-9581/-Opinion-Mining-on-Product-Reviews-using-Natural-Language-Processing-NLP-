# 🌍 Opinion Mining on Product Reviews using Natural Language Processing (NLP)

## 📌 Overview

This project is a multilingual sentiment analysis web application developed using Python and Streamlit. It analyzes product reviews and classifies them into Positive, Negative, or Neutral sentiments using multiple NLP techniques.

---

## 🚀 Features

* Sentiment classification (Positive, Negative, Neutral)
* Multilingual support (English, Hindi, Telugu)
* Multiple models: VADER, TextBlob, BERT
* Language detection using langdetect
* Interactive dashboard with visualizations
* CSV/Excel file upload support
* Download results as CSV
* Model accuracy comparison

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Plotly
* TextBlob
* VADER Sentiment
* Transformers (BERT)
* Scikit-learn

---

## ⚙️ How to Run

```bash id="runproj1"
pip install -r requirements.txt
python -m streamlit run app.py
```
